Pytorch基础

Posted on 2019-06-27 | In Program , Pytorch

记录Pytorch中常使用的基础命令

model.train()

  • model.train()：启用 BatchNormalization 和 Dropout

  • model.eval()：不启用 BatchNormalization 和 Dropout

  • model 训练结束后，会被用来进行测试。这时需要加上 model.eval()，否则有输入数据，即使不训练，网络本身也会改变权值。

torch.Tensor.scatter_(dim, index, src)

  • 分散操作， 从张量 src 中按照 index 张量中指定的索引位置写入 self 张量的值

  • 通俗点儿理解，把 x 放入 y 中。在 dim=0 的情况下，放置的规则就是，先取 x 的第一行，然后根据 index 的第一行进行放入，index 中的每个值代表的是要放入 y 中的第几行。比如，[0, 1, 2, 0, 0] 就是 x 的第一行的第一个元素放入 y 第一行的第一个，第二个放入 y 第二行的第二个，第三个放入 y 第三行的第三个，第四个放入 y 第四行的第四个，依次进行即可。取完 x 的第一行，再取第二行，依次执行

torch.where(condition, x, y)

  • 针对 x 而言，如果其中的每个元素都满足 condition，则返回 x 的值；如果不满足 condition，就用 y 的值替换 x 的值，最后返回结果

torch.clamp(input, min, max)

  • 将输入 tensor 截断在 [min, max] 范围内

torch.norm(input, p, dim, out=None, keepdim=False)

  • 返回输入张量给定维度 dim 上的 p 范数
    • keepdim: 保持输出维度一致
