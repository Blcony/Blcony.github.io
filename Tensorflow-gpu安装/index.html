Tensorflow-gpu安装

Posted on 2019-02-20 | In Tools , Ubuntu

介绍Ubuntu系统下Tensorflow-gpu(1.3)版本的安装方法

环境

  • Ubuntu 16.04
  • Nvidia显卡驱动

安装过程

下载相关软件

  1. 下载cuda8.0(必须是8.0版本)。官网一般是最新版本，所以直接搜索cuda8.0即可。可以下载deb版本。
  2. 下载cudnn v6 for linux(压缩包版本)。

安装cuda8.0

安装之后还要把cuda添加到环境变量中

安装cudnn v6

将下载的压缩包解压，得到cuda文件夹。然后执行一下命令：

安装tensorlflow-gpu
