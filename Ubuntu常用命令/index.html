Ubuntu常用命令

Posted on 2019-04-22 | In Tools , Ubuntu

介绍Ubuntu系统下常用的一些命令

环境

  • Ubuntu 16.04

Apt-get命令

  • apt-get update: 更新本地软件包索引
  • apt-get install: 安装指定软件包，deb文件的下载目录在/var/cache/apt/archives
  • apt-get upgrade: 升级系统中的软件包
  • apt-get remove: 删除系统上的指定软件包，但该操作不会删除软件包的配置文件
  • apt-get purge: 删除软件包及其配置文件
  • apt-get autoremove: 删除所有自动安装且不再使用的软件包（慎用）
  • apt-get dist-upgrade: 升级Linux内核以及没有升级的软件包

解压缩命令

  • zip -r archive_name.zip directory_to_compress: 压缩命令
  • unzip archive_name.zip: 解压命令

删除命令

  • rm -rf directory_to_delete: -r 是向下递归，无论多少级一并删除; -f 是强制删除且没有提示

链接命令

  • sudo ln -s /tmp/filename /usr/local/bin/filename: 建立同步软链接，可以保持链接文件的同步性
  • sudo ln /tmp/filename /usr/local/bin/filename: 建立同步硬链接。与软连接不同的是，硬链接会在你选定的位置上生成一个与原来文件大小相同的文件

查看储存空间命令

  • df -h: 查看分区情况
  • du -h –max-depth=0: 查看当前目录已使用的总空间大小
  • du -h –max-depth=1: 查看当前目录已使用总空间大小以及当前目录下一级文件和文件夹各使用空间大小

更改权限命令

  • chmod [-cfvR][–help][–version] mode file
    • -c: 若该档案权限确实已经更改，才显示其更改目标
    • -f: 若该档案权限无法被更改也不要显示错误讯息
    • -v: 显示权限变更的详细资料
    • -R: 对当前目录下的所有档案与子档案进行相同的权限变更
    • –help: 显示辅助说明
    • –version: 显示版本
    • mode: 权限设定字串，格式如 [ugoa][[+-=][rwx]]
      • u: 表示该档案拥有者
      • g: 表示该档案拥有者属于同一群体(group)者
      • o: 表示其他以外的人
      • a: 表示以上三者都是
      • +: 表示增加权限
      • -: 表示取消权限
      • =: 表示唯一设定权限
      • r: 表示可读取
      • w: 表示可写入
      • x: 表示可执行
  • chmod 命令可以使用数字来表示权限，语法为 chmod abc file
    • a, b, c各为一个数字，表示user, group, other
    • r=4, w=2, x=1
    • chmod a=rwx file <=> chmod 777 file; chmod ug=rwx,o=x file <=> chmod 771 file
