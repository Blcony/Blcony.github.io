Ubuntu无法登录

  1. 重启电脑。

  2. 选择Ubuntu的高级模式。

  3. 选择最新版本的recovery mode，选择resume，从而进入了登录输入密码界面。

  4. 出现问题，输入密码后一直重复输入密码，该mode下的循环登录情况与正常登入时卡在登录界面应该是互相对应的。

环境变量配置

profile 和 bashrc

配置环境变量一般在这两种文件中：

  • profile在系统登录后执行，只在登录系统时执行一次，包括针对系统的 /etc/profile和针对用户的~/.profile。
  • bashrc在每次启动bash shell（打开终端或者在终端输入sh）后执行，包括针对系统的/etc/bash.bashrc和针对用户的~/.bashrc

环境变量

profile的作用

登录shell随着用户的登录而登录，可以看作是第一个shell，后续的shell都是登录shell的子shell。

登录shell会执行包括针对系统的 /etc/profile和针对用户的~/.profile。为了让环境变量在后续的shell都能访问到，可以在这里配置全局的环境变量，但是要注意的是profile只会在登录的时候执行一次，所以一般配置完后需要重新登录才能生效。

bashrc的作用

每次启动shell进程bashrc都会执行，所以一般也可以在这里配置环境变量。

最常用的方法就是vim ~/.bashrc，然后在最后几行加上环境变量的配置，然后source ~/.bashrc或者重启终端即可。
