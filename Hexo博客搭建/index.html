Hexo博客搭建

Posted on 2019-02-16 | In Tools , Github

介绍基于Github的Hexo博客搭建，以及简洁的Next主题优化。

背景

Hexo是基于Node.js的静态博客程序，可以方便的托管到Github上。Hexo的任务是将与内容相关的md文件放在本地，更新博文目录和相关链接信息，调用命令批量完成相关页面的生成，然后将改动后的页面提交到github。

添加公钥

打开 ~/.ssh/ 下建立的密钥文件id_rsa.pub，因为.ssh文件夹是隐藏文件夹，所以用命令行打开比较方便：

点击github首页右上角头像，选择Settings，选择左边栏SSH and GPG keys，再选择New SSH KEY，把打开的id_rsa.pub文件中的密钥复制进去生成公钥。

创建项目仓库

点击github首页右上角+图标，选择New repository。在页面里输入username.github.io（必须这么写）。Readme可以按照自己意愿填写。填完之后点击Create repository。
